# AI Code Explainer

🚀 **AI-powered tool that explains and debugs your code from screenshots.**

যেকোনো প্রোগ্রামিং কোডের screenshot বা ছবি আপলোড করলেই এই টুল স্বয়ংক্রিয়ভাবে কোড extract করে, ভুল চিহ্নিত করে, সমস্যার মূল কারণ ব্যাখ্যা করে এবং সঠিক সমাধান সহ corrected code দিয়ে দেয়।

### Live Demo
[🔗 Try the App](https://first-ai-code-explainer.streamlit.app/)

---

### ✨ Features

- **Image-based Input** — কোড copy-paste করার দরকার নেই, শুধু screenshot আপলোড করলেই চলে
- **Intelligent Code Analysis** — Syntax error, logical issues, runtime problems এবং best practice violations চিহ্নিত করে
- **Step-by-Step Explanation** — কেন এই সমস্যা হয়েছে তা সহজ ভাষায় বিস্তারিত ব্যাখ্যা
- **Corrected Code** — Clean, fixed এবং improved code snippet প্রদান করে
- **Multi-language Support** — Python, JavaScript, Java, C++, React এবং আরও অনেক ল্যাঙ্গুয়েজে কাজ করে
- **Fast & User-friendly Interface** — Streamlit দিয়ে তৈরি সহজ এবং responsive ড্যাশবোর্ড

### 🛠️ Tech Stack

- **Framework**: Streamlit (Python)
- **AI & Vision**: Advanced Gemini Models + Intelligent Prompt Engineering
- **Deployment**: Streamlit Cloud

### 📸 How to Use

1. ওয়েবসাইটে যাও → [AI Code Explainer](https://first-ai-code-explainer.streamlit.app/)
2. তোমার কোডের screenshot আপলোড করো
3. "Explain" বা "Debug" বাটনে ক্লিক করো
4. কয়েক সেকেন্ডের মধ্যে বিস্তারিত explanation + fixed code পেয়ে যাবে

### 🎯 Who is it for?

- Junior Developers
- Students
- Anyone who works with AI-generated code
- Developers যারা debugging-এ অনেক সময় নষ্ট করেন

### 📈 Impact

- Debugging সময় মিনিট থেকে সেকেন্ডে নামিয়ে এনেছে
- বিশেষ করে জটিল error message এবং AI-generated কোড debug করার ক্ষেত্রে অত্যন্
